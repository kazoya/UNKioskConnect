# **App Name**: KioskConnect

## Core Features:

- Biometric Authentication: Utilize IRIS scanning for secure refugee authentication, with potential upgrades for additional modalities.
- Document Scanning & Capture: Scan and securely transmit refugee IDs, passports, and supporting documents.
- Photo Capture: Capture photos compliant with UNHCR document standards using the in-built camera tool. An AI powered tool that can determine if a new photo is needed, if the previous photo is expired, or doesn't match.
- Secure Document Printing: Enable fast, durable, and high-contrast A4 printing of UNHCR documents.
- Multilingual Interface: Provide on-screen flows in both Arabic and English, with accessibility features.
- Data Integration: Integrate with UNHCR systems via secure APIs, including audit logging and encryption.
- Offline Mode: Queue system to allow users to submit requests or other needs that can be fulfilled while offline. Store updates and information for future syncing.

## Style Guidelines:

- Primary color: Deep blue (#294C60) to convey trust and stability, referencing the UN's brand colors.
- Background color: Light gray (#F0F4F8), a desaturated version of the primary color for a clean interface.
- Accent color: Soft teal (#45B69C) to highlight important actions and information.
- Body and headline font: 'PT Sans', a modern and readable humanist sans-serif for clear communication.
- Use clear and universally understandable icons to guide users through the kiosk interface.
- Implement a clean and intuitive layout optimized for touch interactions on a kiosk screen.
- Subtle animations and transitions to guide users and provide feedback.
